## 技术架构

客户端使用前后端分离架构, ArkUI作为渲染框架. rust作为底层调度网络和文件系统.

### 如何构建

1. 安装DevEco-Studio, rust-lang
2. `make`
3. 使用DevEco构建Hap或执行`make all`

### 如何开发调试

阅读鸿蒙官方开发指南
